penuX / MIMIC Demo

Pathogen class prediction from hospital data.

penuX is a research/demo pipeline built around mimic3.py, designed to auto-discover MIMIC-III and MIMIC-IV roots, build mixed text + medication + early-vitals features, and train a PyTorch multiclass classifier with calibration-aware evaluation.

MIMIC-III / MIMIC-IV PyTorch-first One-file pipeline Calibration outputs
Primary script mimic3.py Single-file demo pipeline for data discovery, ETL, training, and evaluation.
Intended use Research only Explicitly framed in the repo as research/education, not clinical deployment.
Model shape Hybrid NN
Signal types 3 feature groups
Outputs ROC / PR / ECE
Repo netanelcyber/penuX

What the model learns

The README describes a fixed-order multiclass pathogen prediction task using microbiology text, antibiotic exposures, and an early-hour vitals/labs window extracted from MIMIC.

Short categorical text Microbiology descriptors and interpretations mapped into token sequences.
Antibiotic exposure One-hot medication signals summarized at the admission level.
Early vitals / labs A configurable clinical window used for numeric feature construction.

Deep-learning architecture

The repo frames penuX as a PyTorch-first model using Embedding, Conv1d, and LSTM components, fused with numeric features before a multiclass prediction head.

ONE FILE Discovery, ETL, training, evaluation, and plotting live in one script.
No pandas Streaming CSV and gzip parsing keeps the pipeline light for demo use.
Calibration-aware The README explicitly calls out Brier/ECE-style evaluation and reliability plots.

Bilingual research surface

The repository README is structured in both Hebrew and English, which makes penuX a strong fit for a dedicated sub-site rather than a one-line portfolio mention.

Hebrew + English The project narrative, warnings, and documentation are presented bilingually.

Responsible use The README explicitly warns that this is not for clinical decision-making.
